## 🧠 About the Book
The *Competitive Programmer’s Handbook* is a widely respected resource covering **algorithms, data structures**, and **problem-solving techniques** for programming contests.  
It helps build strong fundamentals for competitions like **ICPC**, **Codeforces**, **AtCoder**, and **CSES**.

📖 **Author:** Antti Laaksonen  
🌐 **Free PDF:** [Download from CSES.fi](https://cses.fi/book.html)
